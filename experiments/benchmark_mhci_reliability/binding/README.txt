

Cross-validation and blind data sets used in the study are in this folder.

Folder descriptions
===================
./bd2009.1/  
    BD2009  
    Binding data prepared in year 2009.
    Also contains cross-validation data sets: [cv_rnd, cv_sr, cv_gs]

./bd2013.1/  
    BD2013  
    Binding data prepared in year 2013.

./blind.1/   
    Contains the 'Blind' data set for which blind predictions were made using models trained on bd2009.
